The table contains all textual witnesses of the Decretum Burchardi. The list can be narrowed down using the available filters: by Type (e.g. full manuscript, fragment, excerpt, separate transmission), by the Date slider (date of origin), and by the search fields within the table (e.g. signature or place of origin). All filters can be combined. When a digitisation is available, a book icon appears before the manuscript name and links directly to the digitisation. The TEI-XML description of a manuscript can be downloaded directly from the table using the download icon in the Export column. In addition, the data of all manuscripts can be downloaded together as a single JSON file via the Export manuscripts data button. Manuscripts with a complete description are displayed as clickable links in the table and lead to a dedicated manuscript page. There, users can find further information as well as the manuscript’s digitisation. From this page, users can select a book that has already been transcribed and add it to the Workspace, where they can continue working with the text, the digitisation, and additional information.

Below the table, a map view displays the manuscripts’ current storage locations. Using the button at the top right of the map, the visualisation can be switched to the place of origin. Information on the place of origin is available only for manuscripts with a complete description, and therefore only these manuscripts are shown in this view. The colour of the markers indicates the manuscript type (e.g. full manuscript, fragment, excerpt, separate transmission); the Type dropdown above the map allows users to filter the markers accordingly. Clicking on a marker opens a popup with information about the manuscript. For manuscripts with a complete description, the popup also includes a clickable link leading to the manuscript’s dedicated page.

Im Arbeitsbereich werden die sukzessiv erarbeiteten Transkriptionen und später die kollationierten Texte der in Worms entstandenen Handschriften Buch nach Buch veröffentlicht. Aktuell stehen Transkriptionen von Büchern aus folgenden Handschriften zur Verfügung:

  • Bamberg, Staatsbibliothek, Can. 6 (B)
  • Frankfurt am Main, Universitätsbibliothek Johann Christian Senckenberg, Barth. 50 (F)
  • Köln, Erzbischöfliche Diözesan- und Dombibliothek, Cod. 119 (K)
  • Doppelcodex Vatikan, Biblioteca Apostolica Vaticana, Pal. lat. 585 (Va2) und Pal. lat. 586 (Va3).

Der Arbeitsbereich ist Ihr digitaler Schreibtisch, auf dem Sie die verschiedenen Bücher des Decretum Burchardi in verschiedenen Handschriften nebeneinander betrachten und vergleichen können.

Die Schaltfläche Arbeitsmaterial enthält verschiedene Optionen und Funktionen. Über einen Klick auf Ressource hinzufügen, können die Transkriptionen der Bücher auf die virtuelle Arbeitsfläche hinzugefügt werden. Jedes Buch ist angereichert mit einer Transkription, einer strukturierten Inhaltsansicht, digitalisierten Bildern der Handschrift sowie Metadateninformationen. Nach Auswahl der Handschriften erscheint die Schaltfläche Transkriptionsansichten, mit der Sie die Darstellung des Textes ändern und zwischen der ursprünglichen Textstruktur und einer vereinfachten Lesefassung auswählen können. Sie haben zudem die Möglichkeit, Abkürzungen und Schreiberkennzeichnungen ein- oder auszublenden. Es können parallel mehrere Handschriften geöffnet werden, um einen synoptischen Vergleich der Transkriptionen zu ermöglichen.

Zur Orientierung in den Transkriptionen steht eine Legende zur Transkription zur Verfügung. Sie erläutert die im Text verwendeten Farben, Zeichen und visuellen Markierungen (z. B. Initialen, Marginalien, Korrekturen oder Verweise auf das Digitalisat). Die Legende kann entweder über die Schaltfläche Arbeitsmaterial oder über das grüne Info-Icon neben dem Handschriftennamen geöffnet werden.

Über die Option JSON herunterladen im Menü Arbeitsmaterial können Sie Ihren aktuellen Arbeitsstand speichern. Detailliertere Informationen zur JSON-Datei sowie die Download-Funktion selbst finden Sie direkt unter diesem Menüpunkt.

Bitte beachten Sie: Noch stehen nicht alle Transkriptionen zur Verfügung. Abonnieren Sie unseren Newsletter, um über neue Veröffentlichungen informiert zu bleiben.

In the Workspace the transcriptions are published book by book as they are completed, followed by the collated texts of the manuscripts written in Worms. Currently, transcriptions from the following manuscripts are available:

  • Bamberg, Staatsbibliothek, Can. 6 (B)
  • Frankfurt am Main, Universitätsbibliothek Johann Christian Senckenberg, Barth. 50 (F)
  • Köln, Erzbischöfliche Diözesan- und Dombibliothek, Cod. 119 (K)
  • Doppelcodex Vatikan, Biblioteca Apostolica Vaticana, Pal. lat. 585 (Va2) und Pal. lat. 586 (Va3).

The Workspace is your digital desk, where you can view and compare different books of the Decretum Burchardi across various manuscripts side by side.

The Work material button includes various options and functions. By clicking on Add Resource, you can add transcriptions of books to the virtual workspace. Each book comes with a transcription, a structured content view, digitized images of the manuscript, and metadata information. Once the manuscripts are selected, the Transcription Views button appears, allowing you to change the text display and switch between the original text structure and a simplified reading version. You can also choose to show or hide abbreviations and scribal annotations. Multiple manuscripts can be opened simultaneously to enable a synoptic comparison of the transcriptions.

To help you navigate the transcriptions, a Transcription legend is available. It explains the colours, symbols, and visual markers used in the text (e.g. initials, marginalia, corrections, or references to the digital copy). The legend can be opened either via the Work material button or via the green info icon next to the manuscript title.

The Download JSON option in the Work material menu allows you to save your current workspace state. More detailed information about the JSON file and the download itself can be found directly under this menu item.

Eine JSON-Datei ist eine Datei, die strukturierte Daten speichert. In diesem Workspace enthält sie Ihren aktuellen Arbeitsstand: welche Bücher geöffnet sind und welche Ansichten pro Tafel aktiv sind (z. B. Transkription, Struktur, Digitalisat, Info). Damit können Sie Ihre Arbeitsumgebung sichern, später wieder laden oder mit anderen teilen. Um eine gespeicherte Datei erneut zu verwenden, öffnen Sie Arbeitsmaterial, Ressource hinzufügen und laden Sie die Datei im Bereich JSON hochladen hoch.

A JSON file is a file used to store structured data. In this Workspace it records your current state: which books are open and which views are enabled on each panel (e.g. transcription, structure, digital copy, info). This allows you to save your workspace, reload it later, or share it with others. To reuse a saved file, open Work material, Add Resource and upload it in the Load JSON section.
